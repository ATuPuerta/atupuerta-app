# ATuPuerta APP

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ ionic serve
```


```bash
$ npx cap add android
$ npx cap open android

$ ionic build
$ npx cap copy android
$ cd ./android/
$ ./gradlew assembleDebug
$ cd ..
$ adb install .\android\app\build\outputs\apk\debug\app-debug.apk 
```
